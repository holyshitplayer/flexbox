order

По умолчанию flex-элементы располагаются в исходном порядке. Однако с помощью свойства order можно изменить порядок, в котором они появляются в flex-контейнере.

.item { order: 5; /* default is 0 */ }

Элементы с одинаковым значением order располагаются в том порядке, в каком они находятся в исходном коде .

flex-grow

Это свойство определяет способность flex-элемента увеличиваться при необходимости. Оно принимает значение, которое служит пропорцией. Свойство определяет, какое количество доступного пространства внутри flex-контейнера должен занимать элемент.

flex-grow
1
1
1
flex-grow
1
2
1

Если для всех элементов задать flex-grow: 1, оставшееся пространство в контейнере будет распределено поровну между всеми элементами. Если один из элементов имеет значение flex-grow: 2, этот элемент будет занимать в два раза больше места, чем один из других (или, по крайней мере, попытается).

.item { flex-grow: 4; /* default is 0 */ }

Отрицательные числа недействительны.

flex-shrink

Это свойство определяет способность flex-элемента сжиматься при необходимости.

.item { flex-shrink: 3; /* default is 1 */ }

Отрицательные числа недействительны.

flex-basis

Это свойство определяет размер элемента по умолчанию по основной оси. Размер может быть задан вручную (например, 20%, 5rem и т. д.) или с помощью ключевого слова. Ключевое слово auto использует свойство ширина элемента (что временно выполнялось ключевым словом main-size, пока оно не стало устаревшим). Ключевое слово content означает «размер на основе содержимого элемента» — это ключевое слово еще недостаточно хорошо поддерживается браузерами, поэтому его пока не используют, также как и значения max-content, min-content и fit-content.

.item { flex-basis: ⠀⠀| auto; /* default is auto */ }

Если установлено значение 0, дополнительного пространства вокруг содержимого не будет. Если установлено значение auto, дополнительное пространство рассчитывается на основе значения flex-grow элемента.

flex

Это сокращение для свойств flex-grow, flex-shrink и flex-basis. Второй и третий параметры (flex-shrink и flex-basis) являются необязательными. Значение по умолчанию — 0 1 auto, но если Вы измените его с помощью одного числового значения, например flex: 5, это изменит flex-basis на 0% и значения свойств будут такими: flex-grow: 5; flex-shrink: 1; flex-basis: 0%;.

.item { flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]; }

Рекомендуется использовать это сокращение, а не задавать отдельные свойства. Сокращение разумно устанавливает другие значения.

align-self

Это позволяет переопределить выравнивание элемента по текущей flex-строке по умолчанию (или заданное свойство align-items) для отдельных flex-элементов.

align-items: flex-start
flex-start
flex-start
flex-start
flex-end
flex-start
align-self: flex-end

См. align-items, чтобы понять доступные значения.

.item { align-self: auto | flex-start | flex-end | center | baseline | stretch; /* default is auto */ }

Обратите внимание, что float, clear и vertical-align не влияют на flex-элемент, а если у какого-либо flex-элемента margin в поперечной оси выставлен в auto, то align-self игнорируется.
